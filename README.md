## üîó Cross-Schema Integration

- **Legacy**: Traditional ID-based foreign keys (NUMBER types)
- **NDC PLUS**: Modern UUID-based relationships (RAW(16) types)
- **Legacy**: 17 tables focused on core seismic analysis
